Importante: si estás utilizando NVDA 2017.3 o posterior en Windows 10, por favor deshabilita este complemento.

Lleva a cabo el reconocimiento óptico de caracteres (OCR) para extraer texto de un objeto que sea inaccesible. se utiliza el motor OCR Tesseract. Para realizar el OCR, muévete al objeto en cuestión utilizando la navegación de objetos y pulsa NVDA+r. Puedes configurar el idioma de reconocimiento del OCR yendo al menú Preferencias de NVDA y seleccionando Opciones de OCR.
