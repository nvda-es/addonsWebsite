Esta es una versión mejorada del app module para mIRC que viene con NVDA.

descripción del complemento:

Este complemento es una versión mejorada del App Module para mIRC incluido en NVDA.

  1. El navegador de objetos de NVDA ahora se coloca automáticamente en la ventana de salida de mIRC para hacer más fácil su revisión.
  2. Dos teclas rápidas nuevas (NVDA-8 y NVDA-9) conmutan las opciones de "interrupción de voz para carácteres al escribir" y la "interrupción de voz para la tecla intro" respectivamente.
  3. Adicionalmente, NVDA-Intro e Intro del teclado numérico pondrán el navegador de objetos de NVDA en la última línea de texto en la ventana de salida y la verbalizarán.
  4. Las opciones "anunciar cambios de contenido dinámico", "cursor del sistema mueve al cursor de revisión", "interrumpción de voz al escribir carácteres", e "interrupción de voz para la tecla Intro" ahora se guardan automáticamente en un fichero especial de configuración cuando la ventana de mIRC pierda el foco y se restauran cuando lo reciba de nuevo.

Nota: la opción "anunciar cambios decontenido dinámico" de NVDA se habilita por defecto. Si utilizas un sistema de voz para mIRC tal como Talking IRC o mIRC With Speech, desactiva esta opción con la tecla rápida NVDA-5 para prevenir que las líneas entrantes se verbalicen dos veces.
