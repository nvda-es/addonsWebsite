Object Location Tones

  • Autor: Joseph Lee
  • Descargar Versión estable
  • Compatibilidad con NVDA: de 2017.3 a 2019.3

Después de instalar este complemento y de reiniciar NVDA (para cuando habilites este complemento), según navegues a diferentes controles, escucharás tonos para indicar dónde está localizado el objeto en la pantalla. Para desactivar los tonos de localización del objeto, desinstala o deshabilita este complemento desde el Administrador de complementos.

Notas importantes:

  • Si utilizas Audio Themes 3D o complementos similares, se aconseja deshabilitar el complemento Object Location Tones.
  • Si un control está fuera de pantalla, los tonos no se reproducirán.
