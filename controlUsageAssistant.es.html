Control Usage Assistant

  • Autor: Joseph Lee
  • Descargar versión estable
  • Compatibilidad con NVDA: de 2019.3 a 2020.2

Utiliza este complemento para averiguar cómo interactuar con el control enfocado. Pulsa NVDA+H para obtener un breve mensaje de ayuda sobre la interacción con el control enfocado, como con casillas de verificación, campos de edición y así con otros.
