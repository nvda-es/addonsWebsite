Modo de uso

Este simple complemento hace posible crear diccionarios específicos por aplicación para NVDA. Para utilizar el complemento, ve a la aplicación para la que quieres un diccionario, y pulsa NVDA+control+shift+d. Se abrirá el diálogo normal de diccionarios de NVDA para añadir entradas válidas sólo en la aplicación activa (el nombre de la aplicación debería aparecer en el título del diálogo). Se puede cambiar el atajo predeterminado del complemento mediante el diálogo Gestos de entrada de NVDA: bajo el nodo Configuración, expande el nodo que dice "Muestra el diálogo de diccionario específico de la aplicación".

Orden

NVDA+Control+Shift+d

Actualización automática

Este complemento incluye una función de actualización automática. La comprobación de nueva versión se ejecutará cada vez que se cargue NVDA. Si la quieres, ve a NVDA, Preferencias, Opciones, y en la categoría del complemento, marca la casilla.
